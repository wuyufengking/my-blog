<MemoCard date="2025/12/3" :tags="['设计']">

关于页面美化，我遵循了2个原则：对齐 和 让相同的相同、不同的不同

对齐就是页面采用统一的对齐方式，我这里是左对齐，让全部的元素第一个字符都位于同一条线上

让相同的相同、不同的不同，首先是距离上应该要让相同的距离近，不同的距离远，比如h2标签下的若干段文字，应该相距比较近，同时距离下一个h2标签比较远，在视觉上有明显的划分。

然后是通过标记使得相同的相同、不同的不同。我对于h1~h3标签使用红色并采用斜体,p标签的内容则是黑色。同样的从视觉上一眼区分哪些都是标题，哪些都是正文。

</MemoCard>
